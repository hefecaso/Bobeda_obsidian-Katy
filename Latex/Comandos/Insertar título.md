# Insertar título

## Con un titulo colocado de forma normal

Cuando tenemos en el preámbulo un título especificado:

```Latex
\title{Título}
```

Colocamos:

```Latex
\maketitle{}
```

## Título en una página completa

Entre el documento, colocamos al inicio:

```Latex
\begin{titlepage}

	*

\begin{titlepage}
```

## Título en la misma página del contenido

```Latex
\begin{title}

	*

\begin{title}
```
